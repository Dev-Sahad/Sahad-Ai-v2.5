# ☁️ Render Deployment

## 1️⃣ Connect Repository

Connect your GitHub repository to Render Web Service.

---

## 2️⃣ Add Environment Variables

```bash
Render Dashboard → Environment
```

Add:
- API_KEY
- JWT_SECRET
- MY_PASSWORD
- PORT

---

## 3️⃣ Build Settings

### Build Command

```bash
npm install
```

### Start Command

```bash
node server.js
```

---

## 4️⃣ Auto Deploy

Every GitHub push automatically redeploys the application.
